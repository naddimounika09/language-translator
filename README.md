# 🌍 Language Translator

A web-based Language Translation Tool developed using **Python Flask**, **Google Translate API (Deep Translator)**, **HTML**, **CSS**, **Bootstrap**, and **JavaScript**.

## 📌 Project Description

This application allows users to translate text from one language to another using a simple and user-friendly interface. Users can select source and target languages, enter text, and instantly receive the translated output.

This project was developed as part of the **CodeAlpha Internship Program**.

---

## ✨ Features

* Translate text between multiple languages
* Select source and target languages
* Real-time translation using Google Translate
* Copy translated text to clipboard
* Text-to-Speech support
* Responsive and attractive Bootstrap UI
* Easy-to-use interface

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/naddimounika09/language-translator.git
```

### 2. Navigate to Project Folder

```bash
cd language-translator
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open Browser

```text
http://127.0.0.1:5000
```
